== Description ==

This embed plugin allows you to easily and quickly embed images, music and youtube videos into your blog!

All you need to do is enter the tags as shown in the settings page:

[youtube] = Embed a youtube video

[image] = Embed an image

[music] = Embed a sound file

You can find more information on the Settings --> JR Embed page.


== Installation ==

1. Upload `/jr-embed.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit and read the JR Embed Settings in your Administration Panel
4. Add the tags to your pages
